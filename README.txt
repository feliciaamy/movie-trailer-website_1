How to use this code:
1. Open entertainment_center.py. Here you can add as many movie as you want inside refreshMovies method by creating more Movie instance and add it to the movies list
2. Run the entertainment_center.py with IDLE (python) so that the html file can be created.
3. You will get the fresh_tomatoes.html right in your folder.

Some notes:
1. To change or add details to the movie, open the relevant text file.
Storyline —> movie_storyline.txt
Poster URL —> movie_poster_url.txt
Youtube URL —> movie_youtube_url.txt
Rating —> movie_rating.txt
Rank —> movie_rank.txt

2. If there is any css or javascript that you want to add or edit, you can open fresh_tomatoes.py
